simple precise addresses

\ City_name Word_1 Word_2 Word_3 /
e.g. \ Bengaluru cat apple tomato /

An address in this system consists of two parts:

  1. City_name specifies the city.
    Within the current city - it can be omitted
  2. The last three words specify exactly where in that city.
    They are from a limited list of 1024 easy common words

Steps

  • Point a location to generate the corresponding Wolo code address.
    You may also let the system pick your current location automatically or search for a particular place.
    Or,
  • Enter a Wolo code address and hit 'Decode' to go to that location
